# Задача

Предсказать стоимость автомобиля по его характеристикам, текстовому описанию или картинке. 

**О данных**

# Этапы работы

Тренировочные данные (`train`) и тестовые данные (`test`) обрабатывались паралельно. После обработки получил два датасета:
